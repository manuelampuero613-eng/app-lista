# Organizador Universitario

Proyecto listo para desplegar en Vercel o Netlify.

## Verlo en tu computador
```bash
npm install
npm run dev
```

## Subirlo a Vercel
1. Crea un repositorio en GitHub.
2. Sube esta carpeta completa.
3. Entra a Vercel.
4. Elige **Add New Project**.
5. Importa tu repositorio.
6. Vercel detectará Vite automáticamente.
7. Presiona **Deploy**.

## Subirlo a Netlify
1. Sube esta carpeta a GitHub.
2. Entra a Netlify.
3. Elige **Add new site** > **Import an existing project**.
4. Selecciona GitHub y tu repositorio.
5. Usa estos valores:
   - Build command: `npm run build`
   - Publish directory: `dist`
6. Presiona **Deploy site**.

## Notas
- El micrófono funciona mejor en Chrome o Edge.
- Las notificaciones piden permiso del navegador.
- El asistente IA de esta versión usa lógica local, no una API externa.
